# electron-updater-complete-example
This is Complete implementation of Electron Updater, Progress bar, Dialog box

## Build
Step to Run project:

`npm install`

`npm start`

`npm run exe`

After running above you will get .exe file and Unpack of exe. Just install the EXE .
